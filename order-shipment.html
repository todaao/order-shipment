消化月数の計算方法
消化月数 = 良品在庫 ÷ 月平均出荷数 月平均出荷数 = toB月平均 + EC月平均（海外は含まない） toB月平均 = toB出荷実績のある月のうち最古の1ヶ月を除いた直近3ヶ月の平均 （立ち上がり月の外れ値を除去） EC月平均 = 直近3ヶ月の平均
デフォルトLT 初期値: 3ヶ月（12週）
ヶ月
master.jsonにlead_time_weeksがあるSKUはそちらを優先。全商品の基本は3ヶ月（12週）。仕入れ品など短い場合のみ個別設定
欠品警告バッファ 初期値: 2ヶ月
ヶ月
🔴 緊急 = 消化月数 ≤ LT
🟡 警告 = 消化月数 ≤ LT + N ヶ月
⚪ 監視 = 消化月数 ≤ LT + N + 1ヶ月
過剰在庫閾値 初期値: 12ヶ月
ヶ月
消化月数がこの値を超えると「過剰在庫」タブに表示。1年分以上の在庫が目安
予測誤差警告閾値 初期値: 20%
% 超
発注試算と出荷実績の乖離率。50%超は🔴緊急、20〜50%は🟡警告
